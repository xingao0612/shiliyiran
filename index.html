03
衰老状态判断逻辑
  • 差值 > 5岁：衰老加速
  • 0 < 差值 ≤ 5岁：轻度加速
  • 差值 = 0岁：正常状态
  • 差值 < 0岁：衰老缓慢

预测公式
Biological Age = -10.5589
- 0.2148×RBC + 0.2383×MCV
- 0.0809×MCHC - 0.0643×PLT
- 0.2671×MONO + 0.1504×NEUT
+ 0.1746×EO + 0.2093×BA
+ 0.1145×RDW.CV - 0.0293×MPV
- 0.0618×ALT + 0.1791×AST

临床意义
