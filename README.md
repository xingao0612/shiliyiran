一个基于多元线性回归模型的生物学年龄预测工具，通过23项临床检验指标评估机体的生物学年龄，为健康状态评估提供科学参考依据。

## ✨ 核心功能
- **精准预测**：基于23项核心临床检验指标（血液常规、肝功能、肾功能、血脂）构建多元线性回归模型
- **直观对比**：自动计算生物学年龄与实际年龄的差值，评估衰老状态
- **可视化分析**：通过Chart.js生成年龄对比图表，直观展示分析结果
- **现代化UI**：采用玻璃拟态(Glassmorphism)设计风格，视觉体验优秀
- **响应式布局**：适配桌面端、平板、手机等不同尺寸设备
- **动态交互**：包含平滑动画、表单验证、加载状态等交互效果
- **模型说明**：详细展示计算原理和衰老状态判断逻辑

## 🛠️ 技术栈
### 前端技术
- **基础框架**：HTML5 + JavaScript (ES6+)
- **样式框架**：Tailwind CSS v3（原子化CSS）
- **数据可视化**：Chart.js（图表展示）
- **动画效果**：GSAP（GreenSock Animation Platform）
- **图标库**：Lucide Icons（轻量级矢量图标）

### 算法模型
- 多元线性回归模型（Multiple Linear Regression）
- 基于23项临床检验指标的生物学年龄计算公式
